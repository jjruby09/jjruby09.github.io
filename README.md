# jjruby09.github.io

Personal website for JJ Ruby — Senior Director of R&D at the Houston Astros and Visiting Assistant Professor of Mechanical Engineering at the University of Rochester.

Built with plain HTML, CSS, and JavaScript. Hosted on GitHub Pages.

## Structure

- **index.html** — Home / landing page
- **about.html** — Background, education, career path, and awards
- **work.html** — HED physics, baseball R&D, and press coverage
- **contact.html** — Contact info and links
- **css/style.css** — Site-wide styles
- **js/main.js** — Navigation and UI behavior
